Servo

Base class: Drone

Manages a PWM servo channel

Example Configuration

Servo.new 13
  position 0
  pins OUT0_1
  status 1
  name "Rudder"
  map 30 70 110 150
  centre 8
  $position [@>11.16]
  interval 100
  .publish "map"
  .publish "centre"
  .publish "position"
  .publish "output"
.done
