WaterSensor

Base class: Drone

Manages a simple Water sensor connected to a pair of pins

Usage Guide

Example Configuration

[WaterSensor = 5]
  pins = 4, 34
  threshold = 400
  publish = raw, alarm

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 pins u8 2 Pins for the sensor: {output pin}, {analog pin}
11 raw u32 1 Raw analog reading (0..4095)
12 threshold u32 1 Level over which the alarm is triggered
13 alarm f 1 Alarm value, 0=idle, 1=triggered
