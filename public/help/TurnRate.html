TurnRate

Base class:

Generate a turnRate command based on target vs current heading using a PID controller

Usage Guide

Tank steering: * Sub to target heading * Sub to current heading * Tuning parameters (PID) * Generate turn rate

Example Configuration

TurnRate.new 8
  name "TurnRate"
  PID (f) 0.005 0.0 0.0001
  interval 50
  $target [@>22.14]
  $heading [@>50.8]
  .publish "target"
  .publish "heading"
  .publish "PID"
  .publish "turnRate"
.done
