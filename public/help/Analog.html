Analog

Base class: Drone

Manages a simple Analog sensor connected to an analog input pin

Usage Guide

Reads an analog value on specified pin (range 0..4095) and maps into the range limits[min..max]

Example Configuration

[Analog = 5]
  pins = 34
  limits = 0, 10
  publish = raw, analog

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 pins u8 1 Pin for the analog input
11 raw u32 1 Raw analog reading (0..4095)
12 limits f 2 Min and max values to map the raw reading into
13 analog f 1 Analog value after mapping into min/max range
