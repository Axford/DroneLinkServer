RFM69Telemetry

Base class:

Manages DroneLink telemetry using an RFM69HW radio module

Example Configuration

RFM69Telemetry.new 3
  name "RFM69"
  status 1  // enable
  .sub [@>0.0]  // subscribe to and transmit all traffic on this node
  .publish "RSSI"
.done

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
8 RSSI f 1 RSSI of received packets
