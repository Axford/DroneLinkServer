Digital

Base class: Drone

Manages a simple digital IO input on a specified pin

Usage Guide

Reads a digital value on specified pin and maps into the values specified in limits[low, high]

Example Configuration

[Digital = 5]
  pins = 32
  limits = 0, 1
  publish = input

Pubs (outputs / config)

10 pins u8 1 Pin for the digital input
