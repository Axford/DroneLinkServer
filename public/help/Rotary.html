Rotary

Base class: I2CBase

Manages a Adafruit NeoRotary quad encoder input module. Initial values can be set in config.

Example Configuration

[ ROTARY = 14 ]
  name = ROTARY
  bus = 4
  map = 1, 2, 10, 0.1
  input1 = 20
  publish = input1, input2, input3, input4, map

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 input1 f 1 Rotary input 1
11 input2 f 1 Rotary input 2
12 input3 f 1 Rotary input 3
13 input4 f 1 Rotary input 4
