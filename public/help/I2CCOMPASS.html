I2CCompass

Base class: I2CBase

Manages a generic I2C Compass, to be implemented in a concrete class e.g. I2CCompass

Example Configuration

[I2CCompass = 6]
  name="Compass"
  interval= 50
  calibX= -6,0,6
  calibY= -6,0,6
  calibZ=-6,0,6
  trim= 180
  location= -1.8, 52, 100
  mode=1
  centre = -3.188, 4.167, -0.1043
  $location = @>GPS.location
  $roll = @>MPU6050.roll
  $pitch = @>MPU6050.pitch
  publish =heading, vector, calibX, calibY, calibZ
  publish = trim, mode, roll, pitch, raw

Subs (inputs)

Address Addr Address Name Type # Description
16 17 location f 2 Current location from GPS
22 23 pitch f 1 Pitch from IMU (in degrees)
24 25 roll f 1 Roll from IMU (in degrees)

Pubs (outputs / config)

Address Name Type # Description
10 vector f 4 Magnetic field vector after pitch/roll compensation
11 heading f 1 Heading adjusted for magnetic declination
12 declination f 1 Current declination value
13 calibX f 3 Min, center and max magnetic readings for the X axis
14 calibY f 3 Min, center and max magnetic readings for the Y axis
15 calibZ f 3 Min, center and max magnetic readings for the Z axis
18 trim f 1 Manual calibration value to adjust heading to match hull (e.g. for a misaligned physical mount)
20 mode u8 1 Mode: 0=online calibration, 1=fixed calibration, 2=reset calibration, 3=store calibration
21 raw f 4 Raw magnetic field vector
