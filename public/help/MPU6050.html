MPU6050

Base class: I2CBase

Manages a MPU6050 I2c gyro/accel sensor

Example Configuration

[MPU6050 = 14 ]
  name = "MPU6050"
  bus = 4
  interval = 50
  publish = accel, pitch, roll
  publish = raw, calibX, calibY, calibZ, mode

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 accel f 3 Acceleration vector in G
11 gyro f 3 Gyro acceleration vector
12 temperature f 1 Temperature in degrees C
13 pitch f 1 Pitch angle in degrees, negative values are pitch down
14 rol f 1 Roll angle in degrees, negative values are roll right
15 raw f 4 Raw acceleration vector, 4th value is std dev of rolling window
16 calibX f 3 Min, center and max magnetic readings for the X axis
17 calibY f 3 Min, center and max magnetic readings for the Y axis
18 calibZ f 3 Min, center and max magnetic readings for the Z axis
19 mode u8 1 Mode: 0=online calibration, 1=fixed calibration, 2=reset calibration, 3=store calibration
