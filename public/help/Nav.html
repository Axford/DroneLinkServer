Nav

Base class: Drone

Managen navigation between waypoints accounting for crosswind drift

Usage Guide

Target waypoints are defined in terms of Lon Lat and target Radius in meters.

Example Configuration

Nav.new 9
  name "Nav"
  status 1
  interval 50
  $location [@>7.8]
  target -1.7458734889 51.541622044 5
  crosswind 0.2
  correction 20
  wind 0
  .publish "target"
  .publish "location"
  .publish "heading"
  .publish "distance"
  .publish "mode"
  .publish "last"
  .publish "home"
  .publish "crosstrack"
  .publish "correction"
  .publish "wind"
  .publish "crosswind"
  .publish "adjHeading"
.done
