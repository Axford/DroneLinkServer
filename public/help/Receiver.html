Receiver

Base class: Drone

Reads PWM values from an RC receiver

Usage Guide

PWM values from an RC receiver are decoded using interrupt routines, then converted and published as values (value1, value2, etc) in range -1..1 on a regular interval. Output values can be subscribed to from other modules (e.g. Servo or TankSteer).

The valid range of PWM input values (i.e. pulse timings in ms) are defined by the limits parameter.

Deadman protection is triggered after 5 seconds of no valid PWM signals and all output values are set to zero.

Example Configuration

Receiver.new 14
  name "Receiver"
  interval 200
  pins IN0_0 IN0_1
  limits 980 2020
  //.publish "pins"
  .publish "value1"
  .publish "value2"
  .publish "output"
.done
