Controller

Base class: I2CBase

Manage a universal remote control using an SSD1306 display as UI

Example Configuration

[Controller= 10]
  name= "Controller"
  bus= 0

Subs (inputs)

Address Addr Address Name Type # Description
20 21 select f 1 Select button (e.g. JoyStick right button)
22 23 cancel f 1 Cancel button (e.g. JoyStick left button)
24 25 yAxis f 1 yAxis for controlling menus (e.g. JoyStick right yAxis)
