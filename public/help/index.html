Modules

Drone Base class for Drone modules
Depth Manages a simple Depth sensor (sonar transducer) controlled via Trigger and Echo pins
Diagnostic Manages a Diagnostic 128x64 I2C display (1306 driver)
HMC5883L Manages a HMC5883L I2C Compass
Management Provides overall system management and a pub/sub interface to the DroneModuleManager
Motor Manages a Motor via an H Bridge module using PWM
NMEA Manages a serial NMEA GPS device
Nav Managen navigation between waypoints accounting for crosswind drift
Polar Sail polar calibration module
RFM69Telemetry Manages DroneLink telemetry using an RFM69HW radio module
Receiver Reads PWM values from an RC receiver
Servo Manages a PWM servo channel
SpeedControl Convert a distance value (meters) into a speed control value (0..1)
Status Monitors up to four parameters and publishes a corresponding status scene to a Neopixel module
Timer Emits a configured message on a set interval (WIP)
