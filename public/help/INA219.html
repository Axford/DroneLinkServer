INA219

Base class: I2CBase,I2CBase

Manages an INA219 I2C power monitoring module,Manages I2C joystick using DroneJoystick firmware

Example Configuration

[INA219= 4]
  name= "Power"
  bus= 1
  addr= 64
  interval= 1000
  cells= 2
  threshold= 6.4
  status= 1
  publish =current, power, loadV, alarm, shuntV, busV, cellV,[Joystick= 4]
  name= "LeftJoy"
  bus= 1
  invert=0
  publish = x,y,z,button,invert

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 shuntV f 1 Voltage across shunt
11 busV f 1 Bus voltage
12 current f 1 Current (Amps)
13 power f 1 Power (Watts)
14 loadV f 1 Load voltage
15 cellV f 1 Cell voltage (loadV / cells)
16 alarm u8 1 Set to 1 when alarm triggered, 0 otherwise
17 cells u8 1 Number of cells (e.g. for a liPo pack)
18 threshold f 1 Threshold voltage below which alarm is triggered (loadV < threshold)
19 usage f 1 Cumulative usage in Amp hours
10 xAxis f 1 X axis value
11 yAxis f 1 Y axis value
12 zAxis f 1 Z axis value
13 button f 1 Button value
14 invert u8 4 Invert flags for each axis [x,y,z,button]
