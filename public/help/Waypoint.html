Waypoint

Base class:

Load and manage a series of waypoints from a CSV file

Usage Guide

pub mode 0 = normal, 1 = reload waypoints current waypoint number u8 number of waypoints u8 target - current loaded waypoint sub location xxx

Example Configuration

Waypoint.new x
  name "Waypoint"
  
.done

Subs (inputs)

Address Addr Address Name Type # Description
12 13 location f 3 Location from GPS

Pubs (outputs / config)

Address Name Type # Description
8 mode u8 1 Mode 0=normal, 1=Reload waypoint.csv
9 waypoints u8 1 How many waypoints are loaded
10 waypoint u8 1 Current waypoint number
11 target f 3 Target location of current waypoint
