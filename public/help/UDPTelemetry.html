UDPTelemetry

Base class:

Manages DroneLink telemetry using UDP broadcast over WiFi

Example Configuration

UDPTelemetry.new 2
  name "UDPT"
  status 1 // enable
  port 8007
  broadcast 255 255 255 255
  .sub [@>0.0]
.done

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
10 packets u32 3 Packet counters for sent, received and rejected
11 speed f 3 Packet rates per second for sent, received and rejected
