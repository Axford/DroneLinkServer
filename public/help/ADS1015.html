ADS1015

Base class: I2CBase

Manages a Adafruit ADS1015 4-channel 12-bit ADC. Gain set so that raw 4095 = 4.095V

Example Configuration

[ ADS1015 = 14 ]
  name = ADS1015
  bus = 4
  channels = 1
  limits = 0, 0.80566
  publish = value1, raw
